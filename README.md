# Gomoku AI Game

## Overview
This project implements a **Gomoku board game with an AI opponent** using Java.  
The AI uses the **Minimax algorithm** to analyze moves and choose the best strategy.

## Features
- 9×9 Gomoku board
- Human vs AI gameplay
- AI decision making using Minimax
- Move prioritization and board evaluation

## Technologies
- Java
- Object-Oriented Programming
- Minimax Algorithm
